The eLua architecture

This section presents in-depth details about the design and implementation of eLua. It's meant as a guide mainly for eLua developers, but it contains information that can be also valuable for eLua users. An example is the ROM file system chapter. Also, by reading the overview of the eLua architecture you'll get a good view of the overall structure of eLua, which will help no matter if you're a developer or not.

Before diving into this, go to the downloads section and download the source code of eLua, it's much easier to follow this section if you look at the source code.
